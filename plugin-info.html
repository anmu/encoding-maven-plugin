Plugin Documentation

Goals available for this plugin:

Goal Report? Description
encoding:analyze No Analyzes the project structure. The encoding of files is checked against configuration. If nothing is configured, the standard is 'UTF-8'.
encoding:help No Display help information on encoding-maven-plugin.
Call mvn encoding:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
encoding:report Yes Generates a encoding report of all files in project structure. Of every file, the encoding will be shown.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 2.0
JDK Default target for maven-compiler-plugin version 2.5.1
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>de.saxsys.maven.plugins</groupId>
          <artifactId>encoding-maven-plugin</artifactId>
          <version>0.1</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>de.saxsys.maven.plugins</groupId>
        <artifactId>encoding-maven-plugin</artifactId>
        <version>0.1</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
  <!-- To use the report goals in your POM or parent POM -->
  <reporting>
    <plugins>
      <plugin>
        <groupId>de.saxsys.maven.plugins</groupId>
        <artifactId>encoding-maven-plugin</artifactId>
        <version>0.1</version>
      </plugin>
      ...
    </plugins>
  </reporting>
  ...
</project>
