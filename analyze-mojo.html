Encoding Maven Plugin

  • Last Published: 2013-11-11
  • |
  • Version: 0.1

encoding:analyze

Full name:

de.saxsys.maven.plugins:encoding-maven-plugin:0.1:analyze

Description:

Analyzes the project structure. The encoding of files is checked against configuration. If nothing is configured, the standard is 'UTF-8'.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: validate.

Required Parameters

Name Type Since Description
encoding String - General file encoding.
Default value is: UTF-8.
User property is: encoding.encoding.

Optional Parameters

Name Type Since Description
failOnError boolean - Throws Exception if run fails.
Default value is: true.
User property is: encoding.failOnError.

Parameter Details

encoding:

General file encoding.
  • Type: java.lang.String
  • Required: Yes
  • User Property: encoding.encoding
  • Default: UTF-8

failOnError:

Throws Exception if run fails.
  • Type: boolean
